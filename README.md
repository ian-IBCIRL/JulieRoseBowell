## 🚀 How to Deploy (GitHub Pages)

1. Go to your repository on GitHub
2. Click **Settings**
3. Navigate to **Pages**
4. Under **Source**, select:
   - Branch: `main`
   - Folder: `/ (root)`
5. Click **Save**

Your site will go live within about a minute.

---

## ✏️ How to Update Content

To update the site:

- Edit `index.html` directly in GitHub or locally
- Replace `headshot.png` if updating the photo
- Replace `Julie_Rose_Bowell_CV.pdf` if updating the CV

Then commit your changes and the site will auto-update.
